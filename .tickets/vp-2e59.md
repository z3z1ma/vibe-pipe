# CLI Framework

Build command-line interface for VibePiper operations (init, validate, run, test, docs).

## Tasks
1. Create CLI framework using Typer
2. Implement vibepiper init (project scaffolding)
3. Implement vibepiper validate (validate pipeline)
4. Implement vibepiper run (execute pipeline)
5. Implement vibepiper test (run tests)
6. Implement vibepiper docs (generate documentation)
7. Add configuration file support (TOML)
8. Add environment selection (--env flag)
9. Create project templates
10. Add auto-completion support

## Example Usage
```bash
# Scaffold new project
vibepiper init my-pipeline --template=etl

# Validate pipeline
vibepiper validate my-pipeline/

# Run pipeline
vibepiper run my-pipeline/ --asset=customers --env=prod

# Run tests
vibepiper test my-pipeline/

# Generate docs
vibepiper docs my-pipeline/ --output=docs/
```
